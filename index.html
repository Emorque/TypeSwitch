How TypeSwitch Works:

TypeSwitch was created for the Google Gemini Hackathon

Of the available APIs, TypeScript is made solely with Translation API

My goal with TypeSwitch was to create a langauge learning tool that dealt with writing, or in this case, typing

Many langauge learning apps help with recognizing individual words

A key aspect of learning any new language is to learn context

With TypeSwitch, you can input whole sentences, which is more realistic to come across in reading or conversation than individual words
